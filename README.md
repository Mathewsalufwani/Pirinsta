# Pirinsta
#### A clone for the popular social media app, Instagram.
#### By Mathews Alufwani

## Description
As a user, you will be able to;
- Sign in to the application to start using.
- Upload your pictures to the application.
- See my profile with all your pictures.
- Follow other users and see their pictures on your timeline.
- Like a picture and leave a comment on it.


## Live link
Visit the application on https://pirinsta.herokuapp.com/.

## Technologies Used
Technologies used to develop this application:

1. Python v3.6
2. Django
4. HTML 
5. CSS
